# Project-map

原 vouchap-web，已更名为 Project-map。包含**空间与成员**配置及 Mind Map（项目地图）等，共享逻辑已拷贝至 `src/lib/shared`，可单独运行、不依赖 Vouchap 仓库。

- **运行**：`npm run dev`
- **空间/成员**：Space Members、Space Manage、Setup Space、Invite 等页面；逻辑见 `src/lib/shared/space-members.ts` 等
- **Mind Map**：见 [PROJECT_MAP_README.md](PROJECT_MAP_README.md)

若从 Vouchap 多根工作区中移出，可单独打开本目录使用。
